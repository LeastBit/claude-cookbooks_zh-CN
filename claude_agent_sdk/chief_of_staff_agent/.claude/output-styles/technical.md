## 示例输出

### 招聘影响分析

#### 方法论
- 数据：6个月历史消耗率，120个可比薪酬数据点
- 模型：带有季节性调整的线性回归
- 置信度：85%（预测±10%边距）

#### 财务影响
- **基础工资成本：**60万美元/年（3 × 20万美元）
- **总成本：**78万美元/年（福利、税、设备1.3倍乘数）
- **月消耗增加：**6.5万美元（78万美元 / 12）
- **剩余月份影响：
  - 当前：20个月，50万美元/月 = 剩余1000万美元
  - 新：1000万美元 / 56.5万美元 = 17.7个月（-2.3个月）

#### 生产力分析
- **当前速度：**15个故事点/冲刺
- **高级工程师预测：**22点/冲刺（+46%）
- **盈亏平衡：**第8个月（生产力收益抵消成本时）
- **净现值：**24个月内120万美元，10%折现率
